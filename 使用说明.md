# QQ空间相册原图下载工具

## 功能

- 自动获取所有相册列表
- 按 **分类 / 相册名** 自动创建文件夹
- 下载原图（无压缩、无损画质）
- 保存为 JPG 格式到 `E:\qq`
- 支持断点续传（已下载的自动跳过）

## 使用步骤

### 1. 获取Cookie（必须）

1. 用 Chrome 打开 https://user.qzone.qq.com 并登录
2. 按 **F12** 打开开发者工具
3. 切换到 **Network（网络）** 标签
4. 刷新页面，在请求列表中随意点击一个请求
5. 在右侧 **Headers（标头）** 中找到 `Cookie` 字段
6. **右键 - 复制值**，这就是你需要的 Cookie 字符串

### 2. 运行

**方式一：双击 bat 文件（推荐）**

直接双击 `运行下载.bat`，按提示输入QQ号和Cookie。

**方式二：直接编辑脚本**

打开 `qzone_download.py`，在顶部配置区填入：

```python
QQ_NUMBER = "123456789"     # 你的QQ号
COOKIES_STR = "你复制的Cookie内容"
```

然后运行：

```bash
python qzone_download.py
```

**方式三：命令行参数**

```bash
python qzone_download.py 123456789 "Cookie字符串"
```

**方式四：Cookie 文件**

将Cookie保存到同目录下的 `cookies.txt`，然后：

```bash
python qzone_download.py 123456789
```

    └── ...

## 注意事项

- Cookie 有效期通常为 **24小时**，过期需重新获取
- 下载过程中请勿关闭脚本窗口
- 已下载的照片再次运行时会自动跳过
- 如果遇到频繁失败，可能是请求过快被限速，稍后重试即可
- 需要 Python 3.7+ 和 `requests` 库

## 依赖

```bash
pip install requests
```
