# QQ空间相册原图下载工具

自动下载QQ空间相册所有照片，按分类/相册名建文件夹，原图无压缩保存。

## 功能

- 自动获取所有相册列表
- 按 **分类 / 相册名** 自动创建文件夹
- 下载原图（无压缩、无损画质），保存为 JPG
- 每张照片文件名带唯一序号，不会因同名跳过
- 下载完每个相册后校验本地数量与远端是否一致
- MD5 哈希去重检测，自动报告重复照片
- 支持断点续传（已下载的自动跳过）

## 快速开始

### 环境要求

- Python 3.7+
- requests 库

```bash
pip install requests
```

### 双击运行

双击 `运行下载.bat`，自动检测环境并启动。

### 手动配置 Cookie

1. 用 浏览器 打开 https://user.qzone.qq.com 并登录
2. 按 **F12** 打开开发者工具 -> **Network** 标签
3. 刷新页面，点击任意请求，在 **Headers** 中找到 `Cookie` 字段并复制
4. 打开 `qzone_download.py`，填入顶部配置区：

```python
QQ_NUMBER = "123456789"
COOKIES_STR = "你复制的Cookie内容"
```

## 下载校验

每个相册下载完后自动执行：

- **数量校验**：对比 QQ空间显示数量 / API返回数量 / 本地文件数量
- **哈希去重**：计算每张照片的 MD5，报告内容完全相同的重复文件

```
[校验] 相册「生活照」:
  QQ空间显示: 481 张
  API返回:    481 张
  本地已有:   481 张
  本次下载:   481 成功, 0 失败
  [OK] 数量一致
[去重] 发现 2 组重复照片 (共 3 张多余):
  MD5 a1b2c3d4e5f6...: photo_0001.jpg, photo_0005.jpg
  MD5 f6e5d4c3b2a1...: photo_0012.jpg, photo_0013.jpg, photo_0020.jpg
```

## 注意事项

- Cookie 有效期通常为 **24小时**，过期后重新扫码或复制即可
- 下载过程中请勿关闭脚本窗口
- 如果遇到频繁失败，可能是请求过快被限速，稍后重试即可
